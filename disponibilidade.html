Cinnamon Edition


A versão mais popular do Linux Mint é a edição Cinnamon. Cinnamon é desenvolvido principalmente para e pelo Linux Mint. É elegante, bonito e cheio de novos recurso.


Xfce Edition


Xfce é um ambiente de desktop leve. Ele não suporta tantos recursos quanto o Cinnamon, mas é mais leve no uso de recursos.


MATE Edition


MATE é um ambiente de desktop clássico. É a continuação do GNOME 2, que foi o desktop padrão do Linux Mint entre 2006 e 2011.


Cinnamon Edition - "EDGE" ISO


Uma imagem ISO “EDGE” também está disponível para a edição Cinnamon. Esta imagem vem com um kernel mais recente para poder suportar os chipsets de hardware mais recentes. Se você não conseguir inicializar as edições regulares, recomendamos que você experimente o ISO "EDGE".
